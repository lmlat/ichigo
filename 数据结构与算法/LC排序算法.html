    # 排序算法的稳定性

    假定在待排序的记录序列中，存在多个具有相同的关键字的记录，若经过排序，这些记录的相对次序保持不变，即在原序列中，r[i] = r[j]，且 r[i] 在 r[j] 之前，而在排序后的序列中，r[i] 仍在 r[j] 之前，则称这种排序算法是稳定的；否则称为不稳定的。

    # 冒泡排序

    思路分析图：

    思路：一边比较一边向后两两交换，将最大值或最小值冒泡到最后一位；

    public static void bubbleSort(int[] arr) {
    int len = arr.length;
    // len-1趟
    for (int i = 0; i < len - 1; i++) {
        for (int j = 0; j < len - i - 1; j++) {
            //两两比较
            if (arr[j] > arr[j + 1]) {
                //交换
                arr[j] ^= arr[j + 1];
                arr[j + 1] ^= arr[j];
                arr[j] ^= arr[j + 1];
            }
        }
    }
}

    经过优化的思路：使用一个变量记录当前轮次的比较是否发生过交换，若没有交换表示已经有序，不再继续排序；

    public static void bubbleSort2(int[] arr){
    int len = arr.length;
    for(int i = 0; i < len-1; i++){
        boolean isSorted = true;
        for(int j = 0; j < len-i-1; j++){
            if(arr[j] > arr[j+1]){
                //交换
                arr[j] ^= arr[j + 1];
                arr[j + 1] ^= arr[j];
                arr[j] ^= arr[j + 1];
                isSorted = false;
            }
        }
        if(isSorted){
            break;
        }
    }
}

    进一步优化的思路：除了使用变量记录当前轮次是否发生交换外，再使用一个变量记录上次发生交换的位置，下一轮排序时到达上次交换的位置就停止比较；

    public static void bubbleSort3(int[] arr) {
    int len = arr.length;
    //记录最后一次交换的位置
    int lastExchangeIndex = 0;
    //无序数列的边界，每次比较只需要比到这里为止
    int sortBorder = len - 1;
    // len-1趟
    for (int i = 0; i < len - 1; i++) {
        boolean isSorted = true;//有序标记
        for (int j = 0; j < sortBorder; j++) {
            //两两比较
            if (arr[j] > arr[j + 1]) {
                //交换
                arr[j] ^= arr[j + 1];
                arr[j + 1] ^= arr[j];
                arr[j] ^= arr[j + 1];
                isSorted = false;//交换元素，表示之前不是有序的
                lastExchangeIndex = j;//更新为最后一次交换元素的位置
            }
        }
        //更新边界值为最后一次交换元素的位置
        sortBorder = lastExchangeIndex;
        //如果是有序的，则不需要在进行排序
        if (isSorted) {
            break;
        }
    }
    System.out.println(Arrays.toString(arr));
}

    # 插入排序

    思路：

    1. 拿到一个数组arr，以第一个元素为基准，从第二个元素开始进行处理；
    2. 若arr[1] >= arr[0]，表示数据是有序的，无需操作，下标自增；若arr[1] < arr[0]，则交换元素位置，下标自减；
    1, 3, 4, 5, 2, 7 // 初始化的情况
0, 1, 2, 3, 4, 5 // 这是index

index=1开始，3大于1，那么index++
index=2，4大于3，那么index++
index=3，5大于4，那么index++
index=4，2小于5，那么交换5和2的位置，并index--

1, 3, 4, 2, 5, 7 // 交换后
0, 1, 2, 3, 4, 5 // 这是index

index=3，2小于4，继续交换，index--

1, 3, 2, 4, 5, 7 // 交换后
0, 1, 2, 3, 4, 5 // 这是index

index=2，2小于3，继续交换，index--

1, 2, 3, 4, 5, 7 // 交换后
0, 1, 2, 3, 4, 5 // 这是index

index=1，2大于1，完事了

    # 希尔排序

    思路：

    • 将数字按照一定的间隔分为多个子数组，每个组分别进行插入排序；
    • 逐渐缩小间隔进行下一轮排序；
    • 最后一轮时，取间隔为 1，也就相当于直接使用插入排序。

    举个例子，对数组 [84, 83, 88, 87, 61, 50, 70, 60, 80, 99] 进行希尔排序的过程如下：

    间隔值：10/2=5

    • 第一遍（10/2 间隔排序）：按照间隔 5 分割子数组，共分成五组，分别是 [84, 50], [83, 70], [88, 60], [87, 80], [61, 99]。对它们进行插入排序，排序后它们分别变成： [50, 84], [70, 83], [60, 88], [80, 87], [61, 99]，此时整个数组变成 [50, 70, 60, 80, 61, 84, 83, 88, 87, 99]
    • 第二遍（5/2 间隔排序）：按照间隔 2 分割子数组，共分成两组，分别是 [50, 60, 61, 83, 87], [70, 80, 84, 88, 99]。对他们进行插入排序，排序后它们分别变成：[50, 60, 61, 83, 87], [70, 80, 84, 88, 99]，此时整个数组变成 [50, 70, 60, 80, 61, 84, 83, 88, 87, 99]。这里有一个非常重要的性质：当我们完成 2 间隔排序后，这个数组仍然是保持 5 间隔有序的。也就是说，更小间隔的排序没有把上一步的结果变坏。
    • 第三遍（2/2 间隔排序，等于直接插入排序）：按照间隔 1 分割子数组，分成一组，也就是整个数组。对其进行插入排序，经过前两遍排序，数组已经基本有序了，所以这一步只需经过少量交换即可完成排序。排序后数组变成 [50, 60, 61, 70, 80, 83, 84, 87, 88, 99]，整个排序完成。

    # 堆排序

    思路：

    1. 用数组构建出一个大顶堆，取出堆顶的元素；
    2. 调整剩下的元素，构建出新的大顶堆，再次取出堆顶的元素；
    3. 循环往复，完成整个排序；

    # 大顶堆&小顶堆

    • 根节点的值 >= 子节点的值，这种堆称为最大堆，或大顶堆；升序；
    • 根节点的值 <= 子节点的值，这种堆称为最小堆，或小顶堆；降序；

    # 构建大顶堆&调整堆

    构建大顶堆的两种方式：

    1. 从0开始，将每个数字依次插入堆中，一边插入，一边调整堆的结构，使用其满足大顶堆的要求；
    2. 将整个数组的初始状态视作一棵完全二叉树，自底向上调整树的结构，使用其满足大顶堆的要求；

    代码：

    /**
 * “下沉”调整
 * @param array 待调整的堆
 * @param parentIndex 要“下沉”的父节点
 * @param length 堆的有效大小
 */
public static void downAdjust(int[] array, int parentIndex, int length) {
    // temp保存父节点值，用于最后的赋值
    int temp = array[parentIndex];
    int childIndex = 2 * parentIndex + 1;
    while (childIndex < length) {
        // 如果有右孩子，且右孩子大于左孩子的值，则定位到右孩子
        if (childIndex + 1 < length && array[childIndex + 1] > array[childIndex]) {
            childIndex++;
        }
        // 如果父节点大于任何一个孩子的值，则直接跳出
        if (temp >= array[childIndex])
            break;
        //无须真正交换，单向赋值即可
        array[parentIndex] = array[childIndex];
        parentIndex = childIndex;
        childIndex = 2 * childIndex + 1;
    }
    array[parentIndex] = temp;
}

/**
 * 堆排序（升序）
 * @param array 待调整的堆
 */
public static void heapSort(int[] array) {
    // 1.把无序数组构建成最大堆
    for (int i = (array.length-2)/2; i >= 0; i--) {
        downAdjust(array, i, array.length);
    }
    System.out.println(Arrays.toString(array));
    // 2.循环删除堆顶元素，移到集合尾部，调整堆产生新的堆顶
    for (int i = array.length - 1; i > 0; i--) {
        // 最后1个元素和第1个元素进行交换
        int temp = array[i];
        array[i] = array[0];
        array[0] = temp;
        // “下沉”调整最大堆
        downAdjust(array, 0, i);
    }
}


public static void main(String[] args) {
    int[] arr = new int[] {1,3,2,6,5,7,8,9,10,0};
    heapSort(arr);
    System.out.println(Arrays.toString(arr));
}

    # 快速排序

    思路：

    1. 从数组中取出一个元素做为基数；
    2. 遍历数组，将比基数大的元素移动到它的右边，比基数小的元素移动到它的左边。遍历完后，数组就被分成了左右两个区域；
    3. 将左右两个区域理解成两个数组，重复前面两个步骤即可。

    public class 快速排序 {
    public static void swap(int[] nums, int i, int j) {
        int t = nums[i];
        nums[i] = nums[j];
        nums[j] = t;
    }

    public static void qsort(int[] nums, int left, int right) {
        if (left < right) {
            int position = getPosition(nums, left, right);
            qsort(nums, position + 1, right);
            qsort(nums, left, position - 1);
        }
    }

    public static int getPosition(int[] nums, int left, int right) {
        int p = nums[left];
        int mark = left;
        for (int i = left + 1; i <= right; i++) {
            if (nums[i] < p) {
                mark++;
                swap(nums, mark, i);
            }
        }
        nums[left] = nums[mark];
        nums[mark] = p;
        return mark;
    }

    public static void main(String[] args) {
        int[] arr = new int[]{1, 3, 2, 6, 5, 7, 8, 9, 10, 0};
        qsort(arr, 0, arr.length - 1);
        System.out.println(Arrays.toString(arr));
    }
}
